# Filters

Signal processing filters adapted for financial time series.

| Indicator | Full Name | Description |
| :--- | :--- | :--- |
| [Bessel](lib/filters/bessel/Bessel.md) | Bessel Filter | Maximally flat group delay filter. |
| [Bilateral](lib/filters/bilateral/Bilateral.md) | Bilateral Filter | Edge-preserving smoothing filter. |
| [BPF](lib/filters/bpf/Bpf.md) | BandPass Filter | 2nd-order IIR BandPass Filter (Cascade HP+LP). |
| [Butter](lib/filters/butter/Butter.md) | Butterworth Filter | Maximally flat frequency response filter. |
| Cheby1 | Chebyshev I Filter | Steeper roll-off with passband ripples. |
| Cheby2 | Chebyshev II Filter | Equiripple in stopband, monotonic in passband. |
| [Elliptic](lib/filters/elliptic/Elliptic.md) | Elliptic Filter | Equiripple in both passband and stopband. |
| [Gauss](lib/filters/gauss/Gauss.md) | Gaussian Filter | Gaussian Smoothing Filter. |
| [Hann](lib/filters/hann/Hann.md) | Hann Filter | Hann Window Smoothing. |
| [Hp](lib/filters/hp/Hp.md) | Hodrick-Prescott | Causal Trend and Cycle decomposition. |
| [Hpf](lib/filters/hpf/Hpf.md) | High Pass Filter | Attenuates signals below cutoff frequency. |
| [Kalman](lib/filters/kalman/Kalman.md) | Kalman Filter | Recursive state estimation filter. |
| [Loess](lib/filters/loess/Loess.md) | Loess Smoothing | Locally Estimated Scatterplot Smoothing. |
| [Notch](lib/filters/notch/Notch.md) | Notch Filter | Band-stop filter (2-pole IIR). |
| [SGF](lib/filters/sgf/Sgf.md) | Savitzky-Golay | Polynomial smoothing filter. |
| [SSF](lib/filters/ssf/Ssf.md) | Super Smoother | Ehlers Super Smoother Filter. |
| [USF](lib/filters/usf/Usf.md) | Ultra Smoother | Ehlers Ultra Smoother Filter. |
| Wiener | Wiener Filter | Optimum signal estimation filter. |
