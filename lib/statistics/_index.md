# Statistics

Statistical analysis tools applied to price/returns.

| Indicator | Full Name | Description |
| :--- | :--- | :--- |
| **[BETA](beta/beta.md)** | Beta Coefficient | Measures the volatility of an asset relative to the overall market. |
| **[BIAS](bias/bias.md)** | Bias | Measures the percentage deviation of signal from its moving average. |
| **[CMA](cma/Cma.md)** | Cumulative Moving Average | Running average of ALL values (Welford's algorithm). No window. |
| **[COINTEGRATION](cointegration/cointegration.md)** | Cointegration | Tests if two or more time series share a long-term equilibrium relationship. |
| **[CORRELATION](correlation/correlation.md)** | Correlation (Pearson's) | Measures the linear relationship between two variables (-1 to +1). |
| **[COVARIANCE](covariance/Covariance.md)** | Covariance | Measures the joint variability of two random variables. |
| **[CUMMEAN](cummean/cummean.md)** | Cumulative Mean | Calculates the cumulative mean from the start of the series, ignoring na values. |
| **[ENTROPY](entropy/entropy.md)** | Normalized Shannon Entropy | Measures the amount of uncertainty or randomness in a data series. |
| **[GEOMEAN](geomean/geomean.md)** | Geometric Mean | nth root of the product of n numbers; useful for growth rates and ratios. |
| **[GRANGER](granger/granger.md)** | Granger Causality Test | Statistical hypothesis test for determining causality between time series. |
| **[HARMEAN](harmean/harmean.md)** | Harmonic Mean | Reciprocal of the arithmetic mean of reciprocals; for rates/ratios. |
| **[HURST](hurst/hurst.md)** | Hurst Exponent | Measures long-term memory and persistence in a time series. |
| **[IQR](iqr/iqr.md)** | Interquartile Range | Difference between 75th and 25th percentiles; measure of statistical dispersion. |
| **[JB](jb/jb.md)** | Jarque-Bera Test | Goodness-of-fit test for normal distribution (checks skewness & kurtosis). |
| **[KENDALL](kendall/kendall.md)** | Kendall Rank Correlation | Measures the ordinal association between two measured quantities (rank correlation). |
| **[KURTOSIS](kurtosis/kurtosis.md)** | Kurtosis | Measures the "tailedness" or peakedness of a probability distribution. |
| **[LINREG](linreg/LinReg.md)** | Linear Regression Curve | Fits a straight line through price data points using least squares. |
| **[MEDIAN](median/Median.md)** | Median | Finds the middle value in a sorted dataset over a lookback period. |
| **[MODE](mode/mode.md)** | Mode | Finds the most frequently occurring value in a dataset over a lookback period. |
| **[PERCENTILE](percentile/percentile.md)** | Percentile | Calculates the value below which a given percentage of observations fall. |
| **[QUANTILE](quantile/quantile.md)** | Quantile | Divides the probability distribution of a variable into continuous intervals with equal probabilities. |
| **[SKEW](skew/Skew.md)** | Skewness | Measures the asymmetry of the probability distribution of a variable. |
| **[SPEARMAN](spearman/spearman.md)** | Spearman Rank Correlation | Measures monotonic association between two ranked variables (Pearson on ranks). |
| **[STDDEV](stddev/StdDev.md)** | Standard Deviation | Measures the amount of variation or dispersion of a set of values. |
| **[SUM](sum/Sum.md)** | Rolling Sum | Kahan-Babuška summation for numerical stability. |
| **[THEIL](theil/theil.md)** | Theil Index | Measures inequality or concentration in a dataset. |
| **[VARIANCE](variance/Variance.md)** | Variance | Measures the average squared difference from the mean. |
| **[ZSCORE](zscore/zscore.md)** | Z-score Standardization | Indicates how many standard deviations an element is from the mean. |
| **[ZTEST](ztest/ztest.md)** | Z-Test | Statistical hypothesis test comparing sample mean to hypothesized population mean. |
