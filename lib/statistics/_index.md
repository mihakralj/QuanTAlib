# Statistics

Statistical analysis tools applied to price/returns.

| Indicator | Full Name | Description |
| :--- | :--- | :--- |
| **[BETA](beta/beta.md)** | Beta Coefficient | Measures the volatility of an asset relative to the overall market. |
| **BIAS** | Bias | Measures the percentage deviation of signal from its moving average. |
| **[CMA](cma/Cma.md)** | Cumulative Moving Average | Running average of ALL values (Welford's algorithm). No window. |
| **COINTEGRATION** | Cointegration | Tests if two or more time series share a long-term equilibrium relationship. |
| **CORRELATION** | Correlation (Pearson's) | Measures the linear relationship between two variables (-1 to +1). |
| **[COVARIANCE](covariance/Covariance.md)** | Covariance | Measures the joint variability of two random variables. |
| **CUMMEAN** | Cumulative Mean | Calculates the cumulative mean from the start of the series, ignoring na values. |
| **ENTROPY** | Normalized Shannon Entropy | Measures the amount of uncertainty or randomness in a data series. |
| **GEOMEAN** | Geometric Mean | nth root of the product of n numbers; useful for growth rates and ratios. |
| **GRANGER** | Granger Causality Test | Statistical hypothesis test for determining causality between time series. |
| **HARMEAN** | Harmonic Mean | Reciprocal of the arithmetic mean of reciprocals; for rates/ratios. |
| **HURST** | Hurst Exponent | Measures long-term memory and persistence in a time series. |
| **IQR** | Interquartile Range | Difference between 75th and 25th percentiles; measure of statistical dispersion. |
| **JB** | Jarque-Bera Test | Goodness-of-fit test for normal distribution (checks skewness & kurtosis). |
| **KENDALL** | Kendall Rank Correlation | Measures the ordinal association between two measured quantities (rank correlation). |
| **KURTOSIS** | Kurtosis | Measures the "tailedness" or peakedness of a probability distribution. |
| **[LINREG](linreg/LinReg.md)** | Linear Regression Curve | Fits a straight line through price data points using least squares. |
| **[MEDIAN](median/Median.md)** | Median | Finds the middle value in a sorted dataset over a lookback period. |
| **MODE** | Mode | Finds the most frequently occurring value in a dataset over a lookback period. |
| **PERCENTILE** | Percentile | Calculates the value below which a given percentage of observations fall. |
| **QUANTILE** | Quantile | Divides the probability distribution of a variable into continuous intervals with equal probabilities. |
| **[SKEW](skew/Skew.md)** | Skewness | Measures the asymmetry of the probability distribution of a variable. |
| **SPEARMAN** | Spearman Rank Correlation | Measures monotonic association between two ranked variables (Pearson on ranks). |
| **[STDDEV](stddev/StdDev.md)** | Standard Deviation | Measures the amount of variation or dispersion of a set of values. |
| **[SUM](sum/Sum.md)** | Rolling Sum | Kahan-Babuška summation for numerical stability. |
| **THEIL** | Theil Index | Measures inequality or concentration in a dataset. |
| **[VARIANCE](variance/Variance.md)** | Variance | Measures the average squared difference from the mean. |
| **ZSCORE** | Z-score Standardization | Indicates how many standard deviations an element is from the mean. |
| **ZTEST** | Z-Test | Statistical hypothesis test comparing sample mean to hypothesized population mean. |
