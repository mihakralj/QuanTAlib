# Statistics

Statistical analysis tools applied to price/returns.

| Indicator | Full Name | Description |
| :--- | :--- | :--- |
| **[BETA](lib/statistics/beta/beta.md)** | Beta Coefficient | Measures the volatility of an asset relative to the overall market. |
| **[BIAS](lib/statistics/bias/bias.md)** | Bias | Measures the percentage deviation of signal from its moving average. |
| **[CMA](lib/statistics/cma/Cma.md)** | Cumulative Moving Average | Running average of ALL values (Welford's algorithm). No window. |
| **[COINTEGRATION](lib/statistics/cointegration/cointegration.md)** | Cointegration | Tests if two or more time series share a long-term equilibrium relationship. |
| **[CORRELATION](lib/statistics/correlation/correlation.md)** | Correlation (Pearson's) | Measures the linear relationship between two variables (-1 to +1). |
| **[COVARIANCE](lib/statistics/covariance/Covariance.md)** | Covariance | Measures the joint variability of two random variables. |
| **[CUMMEAN](lib/statistics/cummean/cummean.md)** | Cumulative Mean | Calculates the cumulative mean from the start of the series, ignoring na values. |
| **[ENTROPY](lib/statistics/entropy/entropy.md)** | Normalized Shannon Entropy | Measures the amount of uncertainty or randomness in a data series. |
| **[GEOMEAN](lib/statistics/geomean/geomean.md)** | Geometric Mean | nth root of the product of n numbers; useful for growth rates and ratios. |
| **[GRANGER](lib/statistics/granger/granger.md)** | Granger Causality Test | Statistical hypothesis test for determining causality between time series. |
| **[HARMEAN](lib/statistics/harmean/harmean.md)** | Harmonic Mean | Reciprocal of the arithmetic mean of reciprocals; for rates/ratios. |
| **[HURST](lib/statistics/hurst/hurst.md)** | Hurst Exponent | Measures long-term memory and persistence in a time series. |
| **[IQR](lib/statistics/iqr/iqr.md)** | Interquartile Range | Difference between 75th and 25th percentiles; measure of statistical dispersion. |
| **[JB](lib/statistics/jb/jb.md)** | Jarque-Bera Test | Goodness-of-fit test for normal distribution (checks skewness & kurtosis). |
| **[KENDALL](lib/statistics/kendall/kendall.md)** | Kendall Rank Correlation | Measures the ordinal association between two measured quantities (rank correlation). |
| **[KURTOSIS](lib/statistics/kurtosis/kurtosis.md)** | Kurtosis | Measures the "tailedness" or peakedness of a probability distribution. |
| **[LINREG](lib/statistics/linreg/LinReg.md)** | Linear Regression Curve | Fits a straight line through price data points using least squares. |
| **[MEDIAN](lib/statistics/median/Median.md)** | Median | Finds the middle value in a sorted dataset over a lookback period. |
| **[MODE](lib/statistics/mode/mode.md)** | Mode | Finds the most frequently occurring value in a dataset over a lookback period. |
| **[PERCENTILE](lib/statistics/percentile/percentile.md)** | Percentile | Calculates the value below which a given percentage of observations fall. |
| **[QUANTILE](lib/statistics/quantile/quantile.md)** | Quantile | Divides the probability distribution of a variable into continuous intervals with equal probabilities. |
| **[SKEW](lib/statistics/skew/Skew.md)** | Skewness | Measures the asymmetry of the probability distribution of a variable. |
| **[SPEARMAN](lib/statistics/spearman/spearman.md)** | Spearman Rank Correlation | Measures monotonic association between two ranked variables (Pearson on ranks). |
| **[STDDEV](lib/statistics/stddev/StdDev.md)** | Standard Deviation | Measures the amount of variation or dispersion of a set of values. |
| **[SUM](lib/statistics/sum/Sum.md)** | Rolling Sum | Kahan-Babuška summation for numerical stability. |
| **[THEIL](lib/statistics/theil/theil.md)** | Theil Index | Measures inequality or concentration in a dataset. |
| **[VARIANCE](lib/statistics/variance/Variance.md)** | Variance | Measures the average squared difference from the mean. |
| **[ZSCORE](lib/statistics/zscore/zscore.md)** | Z-score Standardization | Indicates how many standard deviations an element is from the mean. |
| **[ZTEST](lib/statistics/ztest/ztest.md)** | Z-Test | Statistical hypothesis test comparing sample mean to hypothesized population mean. |
