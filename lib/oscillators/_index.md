# Oscillators

Oscillators are technical analysis indicators that fluctuate above and below a centerline or within a set of range boundaries as its value changes over time.

## Indicators

| Indicator | Full Name | Description |
| :--- | :--- | :--- |
| **[AC](ac/ac.md)** | Acceleration Oscillator | Measures acceleration/deceleration of the current market driving force (Awesome Oscillator). |
| **[AO](ao/ao.md)** | Awesome Oscillator | A momentum indicator used to measure market momentum and affirm trends or anticipate possible reversals. |
| **[BBB](bbb/bbb.md)** | Bollinger %B | Normalized position within Bollinger Bands; (Price - Lower BB) / (Upper BB - Lower BB). |
| **[BBS](bbs/bbs.md)** | Bollinger Band Squeeze | Identifies low volatility periods when Bollinger Bands contract inside Keltner Channels. |
| **[CFO](cfo/cfo.md)** | Chande Forecast Oscillator | Measures the percentage difference between the closing price and its N-period linear regression forecast. |
| **[DPO](dpo/dpo.md)** | Detrended Price Oscillator | Removes trend from price by comparing to a displaced SMA, highlighting cyclical patterns. |
| **[FISHER](fisher/fisher.md)** | Fisher Transform | Converts price data into a Gaussian normal distribution to identify price reversals. |
| **[INERTIA](inertia/inertia.md)** | Inertia | Measures trend strength based on the distance of the current price from a linear regression line. |
| **[KDJ](kdj/kdj.md)** | KDJ Indicator | Enhanced Stochastic with K, D, and J lines; J = 3K - 2D provides leading momentum signals. |
| **[PGO](pgo/pgo.md)** | Pretty Good Oscillator | Measures distance of current close from its N-period SMA, normalized by ATR. |
| **[SMI](smi/smi.md)** | Stochastic Momentum Index | Measures distance of current close relative to the midpoint of the high/low range. |
| **[STOCH](stoch/stoch.md)** | Stochastic Oscillator | Compares close to price range over time, identifies overbought/oversold conditions. |
| **[STOCHF](stochf/stochf.md)** | Stochastic Fast | Non-smoothed version of the Stochastic Oscillator (%K and %D). |
| **[STOCHRSI](stochrsi/stochrsi.md)** | Stochastic RSI | Applies the Stochastic formula to RSI values, providing more sensitive signals. |
| **[TRIX](trix/trix.md)** | Triple Exponential Average | Measures percentage rate of change of a triple exponentially smoothed moving average. |
| **[ULTOSC](ultosc/ultosc.md)** | Ultimate Oscillator | Combines short, intermediate, and long-term price action into a single oscillator. |
| **[WILLR](willr/willr.md)** | Williams %R | Measures overbought/oversold levels, inverse of the Fast Stochastic Oscillator. |
